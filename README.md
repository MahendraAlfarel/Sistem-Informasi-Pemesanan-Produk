# SISTEM INFORMASI PEMESANAN PRODUK

Sistem Informasi Pemesanan Produk PT Indo Kimia Abadi adalah aplikasi pemesanan produk yang dibangun untuk memudahkan pelanggan dalam melakukan pemesanan produk tanpa harus datang langsung, sistem informasi pemesanan produk juga memudahkan manajer dan karyawan dalam mengelola data produk, data penjualan, dan melihat hasil transaksi

### Tech Stack
Berikut teknologi yang digunakan untuk membangun website sistem informasi pemesanan produk ini:

- Language: HTML, CSS, PHP, & JavaScript.
- Framework: Boostsrap.
- Database: MySQL.
- Vendor: Midtrans

#### Fitur:

- Mengelola Data Manajer.
- Mengelola Data Admin.
- Mengelola Data Produk.
- Mengelola Data Kategori.
- Mengelola Data Pesanan.
- Mengelola Data Invoice.
- Mengelola Data Laporan Transaksi.
- Melakukan Register User.
- Mengelola Profil User.
- Melakukan Pemesanan Produk.
- Melakukan Pembayaran Online.
